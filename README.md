Overview:
This project analyzes Myntra's e-commerce sales data as part of the MATRIX'25 case study: Leveraging Analytics for Myntra's E-commerce Success. Myntra, a leading fashion platform under Flipkart Group, faces challenges like cart abandonment (49.5% overall) and inventory optimization in a competitive landscape. Using a synthetic dataset of customer transactions, this interactive dashboard uncovers behavioral insights, visualizes key metrics, and delivers actionable recommendations to boost retention, conversions, and ROI.
Built with Streamlit for interactivity and Pandas/Plotly for data viz, the dashboard reveals ₹24.58M in revenue from 9,020 units sold to 8,500 customers (AOV: ₹8,000), while highlighting opportunities like 55.3% abandonment in early exploration phases.

Features:
; Interactive Visualizations: Drill-down charts for abandonment journeys, price sensitivity, and marketing efficiency.
; KPI Cards: Real-time metrics on revenue, AOV, discount efficiency (33.12%), and abandonment rates.
; Demographic Breakdowns: Gender-balanced purchases (20-21% across categories) and age/city-specific discount patterns.
; Strategic Insights: Data-backed recs for personalization, AR innovations, and resource reallocation.
; Exportable Summary: 200-word PDF with findings and projections (e.g., 25% ROI uplift).

Tech Stack
; Backend: Python 3.8+, Pandas for data processing, NumPy for computations.
; Frontend: Streamlit for the dashboard UI.
; Visualization: Plotly for interactive graphs (bars, lines, bubbles).
; Data: Synthetic CSV mimicking Myntra's transaction logs (columns: Customer_ID, Gender, Age, City, Product_Category, Price, Quantity, Purchase_Date, Payment_Method, Browsing_Time_mins, Cart_Abandonment_Flag,             Discount_Applied, Loyalty_Points_Earned).
